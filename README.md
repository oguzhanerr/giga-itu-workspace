# giga-itu-workspace

Agent-powered workspace system for the ITU/Giga BCN tech team. Automates vault maintenance, syncs tools, and provides a shared set of PM skills accessible from the terminal or Claude desktop app.

---

## What's included

### Agents
| Agent | Description | Platform |
|---|---|---|
| `housekeeper` | Calendar sync, Meetily export, daily retrospective | macOS (calendar/Meetily), all (assimilate) |
| `product-manager` | PM artefacts, ClickUp sync | All |

### PM Skills (slash commands)
`/write-spec` · `/roadmap-update` · `/stakeholder-update` · `/synthesize-research` · `/competitive-brief` · `/metrics-review`

---

## Prerequisites

| Requirement | Notes |
|---|---|
| [Obsidian](https://obsidian.md) | Your vault |
| [Claude Code CLI](https://claude.ai/code) | `~/.local/bin/claude` |
| Python 3.9+ | `python3 --version` |
| Node.js | For Claude desktop MCP — `brew install node` |
| [fswatch](https://github.com/emcrisostomo/fswatch) | ClickUp sync file watcher — `brew install fswatch` (macOS/Linux) |
| [Meetily](https://meetily.ai) | macOS only, optional |

---

## Setup

```bash
git clone https://github.com/oguzhanerr/giga-itu-workspace
cd giga-itu-workspace
python3 system/install.py
```

The installer will walk you through each component step by step and write your personal `system/config.yaml`.

---

## Structure

```
system/
  install.py               → interactive installer
  requirements.txt         → Python dependencies
  agents/
    housekeeper/           → vault maintenance agent (macOS scheduler)
    product-manager/       → PM skills + ClickUp sync
  utilities/               → one-off scripts
.claude/
  commands/                → slash commands available in Claude Code
```

---

## Configuration

All machine-specific paths live in `system/config.yaml` (generated by installer, not committed). See `system/agents/housekeeper/config.template.yaml` for the full reference.

Key environment variables (can be set instead of config.yaml):

| Variable | Description |
|---|---|
| `VAULT_DIR` | Absolute path to your Obsidian vault |
| `CLAUDE_BIN` | Path to Claude Code CLI |
| `MEETILY_DB` | Path to Meetily SQLite database (macOS) |

## Adding a new agent

1. Create `system/agents/<agent-name>/README.md` with purpose, skills, and setup
2. Add scripts under `system/agents/<agent-name>/`
3. Add slash commands to `.claude/commands/` if applicable
4. Register schedules via the installer or manually
